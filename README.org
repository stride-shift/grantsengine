* Read First
This is the navigation hub. New to the repo? Read this, then ARCHITECTURE.org.

d-lab Grant Engine is a React + Express app that manages a grant funding
pipeline for d-lab NPC (a South African youth digital-skills NPO). It uses the
OpenAI API (proxied through the Express backend) for proposal drafting, funder
research, grant scouting, and pipeline intelligence; the server-side scraper
additionally uses the Anthropic Claude API for parsing/verification.

* Quick Start
#+BEGIN_EXAMPLE
npm install
npm run dev      # concurrently: Express on :3001 + Vite dev server
npm run seed     # seed Supabase with initial pipeline data
npm test         # vitest run
npm run build    # vite production build
npm start        # production: serve dist/ from Express
#+END_EXAMPLE

Requires a =.env= with Supabase credentials, =OPENAI_API_KEY=, and (for the
scraper) =ANTHROPIC_API_KEY=.
