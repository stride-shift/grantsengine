* What This Is

The d-lab Grant Engine is a React + Express grant-pipeline application for d-lab
NPC (The Field Lab NPC), a South African NPO training unemployed youth in
AI-native digital skills. It manages a funding pipeline across stages from
scouting to won/lost, and uses the OpenAI API (proxied through the Express
backend) for proposal drafting, funder research, opportunity scouting, and
pipeline intelligence; the server-side scraper additionally uses the Anthropic
Claude API for parsing. The d-lab domain — financial model, funder landscape,
programme types, proposal strategy — lives in =PRODUCT_CONTEXT.org=.

* Quick Start

** Prerequisites

- Node.js 20.19+ or 22.12+
- An OpenAI API key (=OPENAI_API_KEY=) for the AI features; an Anthropic key
  (=ANTHROPIC_API_KEY=) for the scraper
- Supabase PostgreSQL credentials (=DATABASE_URL=, =SUPABASE_URL=,
  =SUPABASE_SERVICE_KEY=)

** Environment

Backend secrets load from the root =.env=. Copy =.env.example= to =.env= and fill
it in — never commit real secrets. The template lists every variable, including
OpenAI/Anthropic keys, =PORT= (backend, defaults to 3001), =CORS_ORIGIN=,
Supabase credentials, Resend email, Google OAuth, =CRON_SECRET=, and the
Playwright service URL/secret.

** Run

| Command | What it does |
|---------|--------------|
| =npm install= | Install dependencies |
| =npm run dev= | Start both processes via =concurrently=: Express API on =http://localhost:3001= and Vite dev server (HMR) on =http://localhost:3000= — *open :3000* |
| =npm run build= | Production frontend build (=vite build=) → =dist/= |
| =npm start= | Production backend (=NODE_ENV=production node server/index.js=) |
| =npm run seed= | Seed initial pipeline data |
| =npm test= | Run the test suite (=vitest run=) |

Vite (:3000) proxies =/api= to the Express backend (:3001), so the frontend's
relative API calls work with both processes running side by side. The backend
talks to Supabase PostgreSQL; the schema lives in =server/schema.sql=.
